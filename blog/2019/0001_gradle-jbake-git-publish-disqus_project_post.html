Faire un blog avec gradle-jbake-git-publish-disqus

16 avril 2019

Le project gradle-jbake-git-publish-disqus est un blog facile à mettre en ligne pour demarrer l’ecriture d’article. Le project gradle-jbake-git-publish-disqus est une application en ligne de commande. Elle utilise gradle pour automatiser le développement, déploiement et versioning avec le plugin org.ajoberstar.git-publish. le blog est généré avec le plugin org.jbake.site. Ce blog s’appuie sur jbake Le blog généré est hébergeable sur un CDN gratuit.
